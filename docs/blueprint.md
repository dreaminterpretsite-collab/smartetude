## Core Features:

- Secure Authentication: Allow users to register and log in securely using email and password.
- Exercise Submission via AI: Enable users to submit exercises by taking a photo or uploading an image, and use AI tool to provide detailed solutions.
- Balance System: Implement a balance system where solving an exercise costs 200 FCFA, deducted from the user's balance.
- Account Refill via Wave: Allow users to refill their balance through Wave payments, validated manually by admins using transaction IDs.
- Admin Panel: Provide a secure interface for admins to validate/reject pending payments and credit user accounts.
- AI Chatbot Support: Integrate an AI chatbot to answer frequently asked questions about the application's functionality, using AI tool for managing the conversation context and providing support.
- Exercise History: Allow users to view all submitted exercises and review the generated solutions.

## Style Guidelines:

- Primary color: Sky Blue (#ADD8E6) to reflect knowledge.
- Background color: Very light cyan (#F0FFFF) to give clarity.
- Accent color: Gold color (#D4AF37) for CTAs and important elements.
- Headline font: 'Playfair Display' (serif) for an elegant, intellectual feel. Note: currently only Google Fonts are supported.
- Body text font: 'PT Sans' (sans-serif) for a clean, readable style. Note: currently only Google Fonts are supported.
- Use 'Lucide-React' library for clear and coherent icons. 
- Responsive design with a sidebar that turns into a sheet menu on mobile, using ShadCN/UI components.
